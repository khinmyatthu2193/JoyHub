# JoyHub

**Teach with Joy. Learn with Confidence. 🌱**

JoyHub is an offline-friendly interactive classroom engagement platform. Teachers can create quizzes, configure a class, spin an animated student selector, reveal hidden question cards, and celebrate answers with encouraging feedback, sounds, and confetti.

## Run locally

```bash
npm install
npm run dev
```

Create a production build with:

```bash
npm run build
```

## Stack

- React and TypeScript
- Vite and Tailwind CSS
- Framer Motion
- Lucide icons
- Browser local storage and Web Audio API
